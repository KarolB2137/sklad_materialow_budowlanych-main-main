Detailed Description

// Ten kod jest programem w C++, który symuluje system zarządzania magazynem materiałów budowlanych. Użytkownik ma kilka opcji do wyboru w menu, takich jak dodawanie, usuwanie, sprawdzanie stanu magazynu i zapisywanie danych do pliku. Wewnątrz każdej opcji użytkownik jest pytany o odpowiednie dane, takie jak nazwa, cena i ilość materiału. Klasy 'Magazyn' i 'Material' są używane do przechowywania i manipulowania danymi magazynu. Program kończy działanie, gdy użytkownik wybierze opcję "Wyjdź".
